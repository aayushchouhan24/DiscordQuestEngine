## Unlock Discord Developer Tools

Discord may block access to DevTools by default.

### Windows

1. Completely close Discord.
2. Open:

```text
%appdata%\discord\settings.json
```

3. Add or set:

```json
"DANGEROUS_ENABLE_DEVTOOLS_ONLY_ENABLE_IF_YOU_KNOW_WHAT_YOURE_DOING": true
```

4. Save the file.
5. Restart Discord.
6. Press:

```text
Ctrl + Shift + I
```

to open Developer Tools.

## Usage

1. Open Discord DevTools.
2. Go to the **Console** tab.
3. Paste the script.
4. Press Enter.

> ⚠️ Only execute code that you understand and trust.
